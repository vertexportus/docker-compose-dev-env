# docker-laravel
Laravel docker-compose basic env

## env config

- ENV=dev
- COMPOSE_API_VERSION
- COMPOSE_CONVERT_WINDOWS_PATHS
- COMPOSE_FILE=docker/compose/base.yaml:docker/compose/postgres/postgres.yaml
- COMPOSE_HTTP_TIMEOUT
- COMPOSE_TLS_VERSION
- COMPOSE_PROJECT_NAME
- DOCKER_CERT_PATH
- DOCKER_HOST
- DOCKER_TLS_VERIFY

### Postgres

- POSTGRES_PASSWORD
- POSTGRES_DB
